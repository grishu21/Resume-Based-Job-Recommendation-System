# Resume-Based Job Recommendation System

## Overview
A Flask-based web application that recommends jobs based on resume analysis, extracted skills, and user preferences.

## Features
- Resume Upload (PDF/DOCX)
- Dynamic Skill Extraction
- Technical & Non-Technical Skill Identification
- Job Recommendation Engine
- Proof Upload System
- Core-Based Job Recommendations
- General-Based Job Recommendations
- Interview Tracking
- Applied Jobs Management

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Bootstrap
- NLP
- spaCy
- Pandas

## Installation

```bash
pip install -r requirements.txt
python app.py
```
